enum Scenes

Alias for all the scenes in the game.

Use this enum instead of the actual number index or name string. This will be maintained through the development of the game, the literal strings or ints, will not.

Valors de les Enumeracions
LOADING 

The Connect to server scene. This is the first you see at the start of the game.

MANAGER 

The Async persistent Manager scene that is running all the time in the background.

MAIN_MENU 

The main menu scene.

LOBBY 

The lobby scene. This is where you connect to a room and select the character.

INGAME 

The in-game scene. Here you play the actual game.

POST_GAME 

The post-game scene. Here you see a resume of the past game.
