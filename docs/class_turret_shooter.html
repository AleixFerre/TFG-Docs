◆ AttackWithPriority()

void TurretShooter.AttackWithPriority ( )
private

This determines the attack priority of the turret.

The turret has to attack first to all minions, then attacks at the first player that entered the trigger. If there's no entities nearby, it rests until a new entitie enters the trigger.
