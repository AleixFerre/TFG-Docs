int PlayerStats.GetXPNeeded ( int  level)
private

This function will return the xp needed for every level.

Pass the CURRENT LEVEL and it will return the xp amount you need to level up to the next one.

Pre: The level MUST BE [1, 14]

Every level needs 100xp more than the previous one, starting with 250xp for level 2.
