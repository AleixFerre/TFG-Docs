Descripció Detallada

This class communicates directly with the shader to draw the graphs.

Performance here is of upmost importance to reduce as much overhead as possible, as we are updating hundreds of values every frame.

◆ InitializeShader()

void Tayx.Graphy.G_GraphShader.InitializeShader ( )

This is done to avoid a design problem that arrays in shaders have, and should be called before initializing any shader graph.

The first time that you use initialize an array, the size of the array in the shader is fixed. This is why sometimes you will get a warning saying that the array size will be capped. It shouldn't generate any issues, but in the worst case scenario just reset the Unity Editor (if for some reason the shaders reload). I also cache the Property IDs, that make access faster to modify shader parameters.
